# 栅格布局
## 基础用法
<ClientOnly>
  <grid-demo-1></grid-demo-1>
</ClientOnly>

## Grid Atrributes
### Row Atrributes
> - gutter:栅格的间隙，单位为 px	,string / number类型
> - align:栅格统一朝向, string类型, 可选值为 left,center,right
### Col Attributes
> - span:栅格的宽度,string / number类型
> - offset:栅格偏移量(向右),string / number类型
> - 适配参数: iPad,narrowPc,pc,widePc
